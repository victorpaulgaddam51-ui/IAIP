# ModernStore Frontend

A modern React + Vite storefront with a premium UI, product search, cart management, and checkout flow.

## Features
- Responsive product catalog
- Search and category filtering
- Shopping cart with quantity controls
- Checkout flow with graceful fallback when the backend is unavailable
- Indian Rupee pricing

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown by Vite.

## Build

```bash
npm run build
```
